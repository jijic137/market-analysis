# CloudBase 云开发部署指南

## 第一步：开通云开发环境

1. 打开浏览器，访问 [腾讯云控制台](https://console.cloud.tencent.com/tcb)
2. 如果未登录，使用微信或 QQ 登录
3. 点击"新建环境"
4. 选择"按量计费"模式（有免费额度，足够个人使用）
5. 填写环境名称（如：market-analysis）
6. 点击"立即开通"
7. 等待环境创建完成（约 1-2 分钟）
8. 记录环境 ID（格式如：env-xxxxxxxx）

## 第二步：开启静态网站托管

1. 在云开发控制台，点击左侧"静态网站托管"
2. 点击"开启"
3. 选择"按量计费"（免费额度充足）
4. 等待开启完成

## 第三步：上传网站文件

### 方式一：控制台上传（推荐新手）

1. 在"静态网站托管"页面
2. 点击"文件管理"
3. 点击"上传文件"
4. 选择以下文件上传：
   - `index.html`（主页面）
   - `history.html`（历史记录页）
   - `styles.css`（样式文件）
   - `app.js`（脚本文件）
   - `README.md`（说明文档）
   
5. 同时上传整个 `data` 文件夹

### 方式二：使用命令行工具

如果您的电脑已安装 Node.js，可以使用命令行工具：

```bash
# 安装 CloudBase CLI
npm install -g @cloudbase/cli

# 登录
tcb login

# 部署（替换 env-xxxx 为您的环境ID）
tcb hosting deploy ./ -e env-xxxx
```

## 第四步：配置默认域名

1. 在"静态网站托管" → "设置"
2. 找到"默认域名"
3. 域名格式为：`您的环境ID.tcloudbaseapp.com`
4. 设置 `index.html` 为默认首页

## 第五步：访问网站

部署完成后，通过以下地址访问：
```
https://您的环境ID.tcloudbaseapp.com
```

例如：`https://env-123456.tcloudbaseapp.com`

## 高级配置（可选）

### 自定义域名

1. 在"设置" → "自定义域名"
2. 添加您的域名
3. 按提示配置 DNS 解析

### HTTPS 强制跳转

1. 在"设置"中开启"HTTPS 强制跳转"
2. 确保网站安全访问

## 免费额度说明

CloudBase 按量计费提供免费额度：

- **静态托管**：5GB 存储 + 5GB/月 流量
- **CDN 流量**：5GB/月
- **足够个人网站使用**

超出额度后按实际使用量付费，费用很低。

## 遇到问题？

### Q: 访问显示 404
A: 检查是否上传了 `index.html`，并设置为默认首页

### Q: 页面样式丢失
A: 确认 `styles.css` 文件已上传

### Q: 图表不显示
A: 网站使用 Chart.js CDN，需要联网访问

## 自动更新配置

网站已配置 WorkBuddy 自动化任务：
- 每天早上 9:00 自动更新数据
- 自动更新 `index.html` 文件
- 保持数据最新

部署后，需要重新上传更新后的文件即可。
