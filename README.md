# 每日金融市场行情分析网站

这是一个自动更新的金融行情分析网站，每日早上9点自动更新最新数据。

## 功能特点

- 📊 A股、美股、港股实时行情
- 📈 交互式走势图表
- 🌍 国际局势分析
- 💡 专业投资建议
- 📱 响应式设计，支持移动端
- 🔄 每日自动更新

## 项目结构

```
Claw/
├── index.html          # 主页
├── history.html        # 历史报告
├── styles.css          # 样式文件
├── app.js              # 主逻辑文件
├── data/
│   ├── market_data.json    # 当前数据
│   └── history/            # 历史数据
├── reports/            # 历史报告存档
└── README.md           # 说明文档
```

## 部署说明

本网站支持多种部署方式：

### 1. GitHub Pages（推荐）
- 免费托管
- 自动HTTPS
- 支持自定义域名

### 2. CloudBase 云开发
- 腾讯云托管
- 国内访问快
- 支持云函数

### 3. Vercel
- 自动部署
- 全球CDN
- 免费额度充足

### 4. Netlify
- 简单易用
- 自动HTTPS
- 免费托管

## 自动更新机制

网站通过 WorkBuddy 自动化任务系统每日更新：

- **更新时间**：每天早上 9:00
- **更新内容**：市场数据、走势图表、投资建议
- **历史保留**：自动保存历史快照

## 数据来源

- 东方财富网
- 新浪财经
- 腾讯证券
- 华尔街见闻
